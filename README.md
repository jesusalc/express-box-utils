# express-box-utils
Box Utils for Express() server in nodejs

  
```shell 
    npm install express-box-utils --save
    npm test
```


  
```Javascript 
    const express-box-utils = require('express-box-utils');
    app.use(express-box-utils.boxed);
```
 
 
```Javascript 
    app.use(require('express-box-utils').boxed);
```
